## 흐름 정리

### QR 직원 체크인
```
MainCheckinScreen → QR 스캔 → onQrScanned(qrData)
  → NavGraph → checkin_qr/{qrData} → CheckinScreen → CheckinViewModel
  → authenticate() → qrLogin API → role=="EMPLOYEE"
  → performEmployeeCheckin() → POST /checkin/employee
  → employeeCheckinDone=true → CheckinScreen LaunchedEffect
  → onEmployeeCheckinComplete(name, time, count)
  → NavGraph → employee_checkin_complete/{name}/{time}/{count}
  → EmployeeCheckinCompleteScreen (5초 후 자동 홈 복귀)
```

### 전화번호 직원 체크인
```
MainCheckinScreen → 전화번호 8자리 입력 → PhoneLoginViewModel
  → login() → phoneLogin API → role=="EMPLOYEE"
  → employeeCheckin API 직접 호출 → employeeCheckinDone=true
  → MainCheckinScreen LaunchedEffect 감지
  → onEmployeeCheckinDone(name, time, count)
  → NavGraph → employee_checkin_complete/{name}/{time}/{count}
  → EmployeeCheckinCompleteScreen (5초 후 자동 홈 복귀)
```

### 3. DB DDL 실행
```sql
ALTER TABLE employee ADD COLUMN QR_CODE VARCHAR(500) NULL COMMENT 'QR 이미지 S3 URL' AFTER EMPLOYEE_LOGIN_YN;
ALTER TABLE employee ADD COLUMN QR_CONFIRMATION_TOKEN VARCHAR(20) NULL COMMENT 'QR 토큰 (10자)' AFTER QR_CODE;

CREATE TABLE employee_check_in_log_history (
    ID              BIGINT          NOT NULL AUTO_INCREMENT,
    BRANCH_ID       BIGINT          NOT NULL COMMENT '지점 ID',
    USER_ID         BIGINT          NOT NULL COMMENT '유저 ID',
    EMPLOYEE_ID     BIGINT          NOT NULL COMMENT '직원 ID',
    CHECK_IN_METHOD VARCHAR(20)     NOT NULL COMMENT '체크인 방식 (QR, PHONE)',
    STATUS          VARCHAR(20)     NOT NULL COMMENT '상태 (APPROVED, ERROR)',
    ERROR_MESSAGE   VARCHAR(500)    NULL     COMMENT '에러 메시지',
    WRITE_DATE      DATE            NOT NULL COMMENT '체크인 날짜',
    WRITE_TIME      TIME            NOT NULL COMMENT '체크인 시간',
    REG_DT          DATETIME        NOT NULL DEFAULT CURRENT_TIMESTAMP,
    UPD_DT          DATETIME        NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    PRIMARY KEY (ID),
    INDEX idx_employee_checkin_branch_date (BRANCH_ID, WRITE_DATE),
    INDEX idx_employee_checkin_employee (EMPLOYEE_ID)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='직원 키오스크 체크인 로그';
```
